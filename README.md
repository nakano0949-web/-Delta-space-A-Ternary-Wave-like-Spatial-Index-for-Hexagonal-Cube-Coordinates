# -Delta-space-A-Ternary-Wave-like-Spatial-Index-for-Hexagonal-Cube-Coordinates
Delta-space is a new ternary, wave-like spatial indexing framework for hexagonal grids. It combines balanced ternary digits with cube-coordinate geometry to build a multi-scale 3-ary tree (Delta-tree) supporting NN, kNN, and range search with provable pruning guarantees.
